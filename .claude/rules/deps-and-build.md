# Dependencies & Build

## Key Dependencies

- `vitepress@^1.6.3` - Static site generator
- `vue@^3.5.13` - UI framework
- `tailwindcss@^4.1.18` - Utility CSS
- `mermaid@^11.11.0` - Diagram rendering
- `markdown-it-mathjax3@^4.3.2` - Math rendering
- `markdown-it-plantuml@^1.4.1` - PlantUML plugin
- `@catppuccin/vitepress@^0.1.2` - Color theme

## Build & Deployment

- **Build Output**: `docs/.vitepress/dist/` (generated by `pnpm docs:build`)
- **Dev Server**: Runs on `http://localhost:5173` by default
- **Static Site**: Can be deployed to any static hosting (GitHub Pages, Vercel, Netlify, etc.)
