# Theme & Styling Customization

- **Colors**: Modify Catppuccin theme in `config.mts` (light/dark)
- **Styles**: Add CSS to `docs/.vitepress/theme/style/index.css`
- **Components**: Add Vue components to `docs/.vitepress/theme/componnets/` and register in `theme/index.ts`
